# machine-learning-knowledge-level-classification-and-summarization-educational-video
Machine Learning based framework for knowledge-level classification and summarization of educational videos using Whisper, BERT and LED proposes an AI-powered framework that automatically analyzes educational videos and transforms them into structured learning insights.

🔹 Speech-to-Text Conversion: Utilized Whisper to generate accurate transcripts from educational videos along with timestamps.

🔹 Knowledge-Level Classification: Leveraged BERT to classify content into four cognitive knowledge categories:
• Factual
• Conceptual
• Procedural
• Metacognitive

🔹 Automated Summarization: Employed LED (Longformer Encoder Decoder) to generate meaningful summaries from lengthy educational transcripts.

🔹 Timestamp-Based Learning Insights: Enabled mapping of knowledge categories to specific video segments, allowing learners to easily navigate and revisit important concepts.

🔹 Structured Educational Analytics: Generated knowledge distributions and categorized content, helping educators and learners better understand the instructional focus of a video.

This work demonstrates how Artificial Intelligence, Natural Language Processing, and Generative AI can be applied to make educational content more accessible, searchable, and learner-centric.
